# Let Me Here

Letmehere is a lightweight block sitting mod.

## Feature

With default keybinds:

- `Z` to **sit**.
- `LCtrl` + `Z` to **sit in the block horizon center**.

## Command

- `/sit [<location>] [<align>]` Make the executor sit in a certain position. 1 block distance limitation is applied to the distance from the executor to the target location when the command source have no gamemaster command permission level.
